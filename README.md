# 🤖 7AMDAAN.IO | Autonomous AdTech Systems

**Revolutionizing the DOOH (Digital Out-of-Home) sector through interactive, autonomous robotic interfaces.**

## 🌐 Overview

7amdaan-v2 is a high-performance web interface designed to showcase the next generation of advertising technology. Built with a "Terminal Green" aesthetic, the platform highlights the integration of AI-driven targeting, voice interaction, and cloud-based fleet management.

**Live URL:** [https://7amdaanio.netlify.app]()

---

## 🛠 Tech Stack

* **Framework:** React 18 (Vite)
* **Styling:** Tailwind CSS
* **Animations:** Framer Motion (System-wide hardware-accelerated transitions)
* **Icons:** Lucide React
* **Deployment:** Netlify (CI/CD via GitHub)
* **Forms:** Formspree Integration

---

## 🚀 Key Features

### 1. High-Impact UI/UX

* **Terminal Aesthetic:** A custom "Robot Dark" theme with `robot-accent` highlights.
* **HUD Accents:** Fixed-position corner accents to simulate a robotic vision interface.
* **Interactive Cursor:** Custom-designed cursor tracking for enhanced user engagement.

### 2. Autonomous Catalog

* **Dynamic Grid:** An adaptive layout showcasing robotics features and case studies.
* **Optimized Video Streaming:** Implementation of `preload="metadata"` and `onCanPlayThrough` logic to handle high-resolution 60MB video assets smoothly.
* **System Initialization Loaders:** Contextual loading states that maintain the robotic brand identity during data buffering.

### 3. Core Protocols (About)

* Detailed breakdown of technical capabilities including **Targeted Ads**, **Voice AI**, and **Cloud CMS** synchronization.

## 📦 Installation & Setup

1. **Clone the repository:**
```bash
git clone https://github.com/TariqAfandi/7amdaan-v2.git
cd 7amdaan-v2

```


2. **Install dependencies:**
```bash
npm install

```


3. **Run development server:**

```bash
   npm run dev

```

4. **Build for production:**
```bash
npm run build

```



---

## 🔧 Performance Optimization

To maintain a high-tech experience despite heavy video assets, the project utilizes:

* **Intersection Observer API:** For lazy-loading components and assets.
* **Bento Grid Logic:** For efficient spatial organization of media.
* **Media Buffering States:** Stylized React state management to prevent "black-box" loading gaps in the gallery.
